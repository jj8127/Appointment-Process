테스트 계획 (프로세스 흐름 기반)

FC 기본 정보 입력

정상: FC 로그인 후 필수 필드(이름/전화/주민번호/주소 등) 저장, 재진입 시 값 유지.
오류: 필수값 미입력, 전화/주민번호 포맷 오류, 저장 실패 시 에러 노출/저장 차단.
보안: 다른 FC 계정으로 로그인 시 데이터 격리.
총무 임시사번 발행 및 경력 등록

정상: 총무가 FC 선택 → 임시사번 생성, 경력 유형(신입/경력) 저장.
검증: 생성 후 FC 프로필/수당동의 화면에 임시사번 표시 여부.
FC 수당 동의 및 날짜 입력

정상: 임시사번 보유 FC가 수당 동의 슬라이드 확인 → 동의일 입력 → 제출 성공.
오류: 임시사번 누락 시 접근/제출 차단, 날짜 미입력 시 검증 에러.
총무 수당 동의 승인 및 필수 서류 목록 등록

정상: 총무가 동의 요청 승인 → FC 상태 변경, 필수 서류 목록(문서 타입별) 생성.
검증: FC 화면에 요청된 서류 리스트 노출 여부.
FC 서류 등록

정상: 각 서류 업로드(이름/타입/파일) → 진행률/상태 표기.
오류: 파일 미선택, 잘못된 형식/용량 초과 시 에러 노출; 업로드 실패 시 재시도 가능.
총무 서류 개별 승인

정상: 총무가 문서별 승인/반려 → FC 화면 상태 업데이트(승인/반려 사유).
실시간/동기화: 승인 후 FC 진행 상태/스텝 전환 확인.
총무 위촉 차수 입력 (생명/손해)

정상: 총무가 차수(예: 12월 1차) 입력/저장.
검증: FC 화면에 차수 표시 여부.
FC 위촉 날짜 입력

정상: FC가 위촉 날짜 입력 후 제출.
오류: 날짜 미입력/과거일자 입력 시 검증.
총무 위촉 날짜 승인

정상: 총무 승인 후 FC 상태 “위촉 완료” 전환, 대시보드 반영.
시험 신청 프로세스

접근 제약: 수당 동의 승인된 FC만 시험 신청 버튼/페이지 접근 가능(비승인 시 차단 메시지).
FC 시험 신청
정상: 시험 타입/일시/장소 등 입력 후 제출.
검증: 제출 후 상태 “신청 완료”로 표시.
총무 시험 등록 완료
정상: 총무가 신청 건 승인/등록 처리 → FC 상태 갱신(예: “등록 완료”).
오류: 승인 실패 시 에러 노출 및 FC 상태 유지.
공통 비기능/검증 포인트

상태 전이: 각 단계 완료 시 대시보드/스텝 표시 정확성.
권한: 총무 기능과 FC 기능 분리, 권한 없는 메뉴 차단.
입력 검증: 전화번호 11자리, 주민번호 앞6+뒤7, 날짜 포맷/필수값 검증.
영속성: 앱 재시작/재로그인 후 데이터/상태 유지.
오류 처리: 네트워크/저장 실패 시 사용자 피드백 및 재시도 가능 여부.
